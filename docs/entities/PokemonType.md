# Pokémon Type

## Description

Representation of a Pokémon type at the application.

## Properties

- id: Unique identifier of the Pokémon type.
- name: Name of the Pokémon type.
- color: Color of the Pokémon type.

## Pseudocode

```typescript
interface Type {
    id: Uuid;
    name: string;
    color: string;
}
```
