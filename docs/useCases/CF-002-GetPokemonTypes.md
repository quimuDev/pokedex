## CF-002 - Get Pokémon Types

## 1. Description

The system retrieves the list of all available Pokémon types to enable filtering in the Pokémon list.

## 2. Main User

User.

## 3. Inputs

Required:

- None.

Optional:

- None.

## 4. Outputs

A list of all available Pokémon types.

## 5. Preconditions

None.

## 6. Main Flow

1. The user accesses the Pokémon list page.
2. The system requests the list of available types.
3. The system returns the list of types.

## 7. Alternative Flows

None.

## 8. Postconditions

None.

## 9. Exceptions

None.
